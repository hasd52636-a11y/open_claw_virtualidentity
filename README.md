# open_claw_virtualidentity

虚拟身份生成系统，用于OpenClaw平台的测试注册场景。

## 功能特性

- 支持多国家/地区的虚拟身份生成
- 基于哈希的API密钥验证
- 完整的身份信息字段
- 多语言支持
- 历史记录管理
- 导出功能（JSON、Excel、复制）

## 技术栈

- 前端：React + TypeScript + Tailwind CSS
- 后端：Node.js + Express + SQLite
- 部署：Vercel

## API文档

详细的API使用说明请查看 `API` 目录下的文档。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 启动后端服务器
npm run server
```

## 部署

项目已配置好Vercel部署，可直接部署到Vercel平台。
